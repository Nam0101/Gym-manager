Ứng dụng quản lý phòng gym

Cách cài đặt:

1. Clone repository về máy:
```
git clone https://github.com/Nam0101/Gym-manager.git
```
2. Cài đặt các thư viện cần thiết:
```
pip install -r requirements.txt
```
3. Chạy chương trình:
```
python manage.py runserver
```
